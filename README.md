# Movement Detector

En base al background substractor GMG, genera máscaras en entornos con luz variable para clasificar videos y detectar si están en movimiento relativo al entorno o no.

## Requirements

* python
* numpy
* cv2
* re
* tqdm

## Usage

```
python3 main.py
```
